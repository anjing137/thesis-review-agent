# 论文评价 Agent

## 简介

本Agent对经济与管理学院本科生学年论文/毕业论文进行专业评价。

**评价维度**（共6个，总分100分）：

| 维度 | 权重 | 说明 |
|-----|------|------|
| 选题与研究问题 | 15% | 题目质量、研究必要性、研究价值 |
| 参考文献与学术规范 | 15% | 数量≥15篇（硬性）、质量、格式、外文≥3篇 |
| 内容创新性 | 15% | 研究视角、方法、发现、文献评述 |
| 框架与逻辑结构 | 20% | 章节完整、逻辑清晰，引言质量 |
| 方法与论证严谨性 | 25% | 理论分析、稳健性检验（强制）、内生性、**过度控制检测** |
| 语言与表达 | 10% | 第三人称、术语规范、标点、图表 |

**新增能力**：对于实证性论文，额外进行**模型设定审查**，自动检测"过度控制"（over-control）等问题。

## 适用场景

- 学年论文评价
- 毕业论文初稿评价
- 学生论文修改指导
- 论文质量检查

## 使用方法

### 单篇论文评价

分两步进行：

**第一步：解析论文并生成评价 prompt**

```bash
python main.py 论文.docx --prompt -o ./reviews
```

输出文件：
- `{论文名}_review_data.json` — 包含完整解析结果（学生信息、正文字数、参考文献统计、表格图片数量等）
- `{论文名}_review_prompt.md` — 供 AI 模型评价使用的 prompt（包含论文摘要、前15000字正文、前5000字参考文献）

**第二步：AI 评价并生成报告**

将第一步生成的 prompt 内容发送给 AI 模型进行评价，AI 直接返回 Markdown 格式评价报告，然后用以下命令保存：

```bash
python main.py 论文.docx --report -e "AI返回的Markdown评价报告" -o ./reviews
```

输出文件：
- `{论文名}_评价报告.md` — 结构化评价报告

### 批量评价

```bash
python main.py --batch ./papers/ -o ./reviews --summary
```

批量处理目录下所有 .doc/.docx 论文，输出每篇论文的 `_review_data.json` 解析结果；加 `--summary` 时额外生成 `summary.csv` 汇总表。

## 评价流程

### 第一步：Python 规则检测

运行 `python main.py 论文.docx --prompt -o ./reviews`，Python 层自动完成：
- docx 文件 XML 解析
- 学生信息提取（姓名、学号、专业、题目、导师）
- 论文类型判断（实证性/学理性）
- 参考文献分析（数量、外文、期刊占比、近五年占比）
- 正文字数统计（中文、英文分开）
- 表格与图片数量统计（区分原生表格与截图）
- 写作规范初检（第一人称统计）

### 第二步：AI 深度评价

Agent读取第一步生成的 `{论文名}_review_prompt.md`，基于论文内容进行评价：
- 逐维度评价（引用原文）
- 计算各维度得分
- 识别优点和问题
- 生成具体可操作的修改建议
- **实证性论文额外进行模型设定审查**
  - 提取变量定义和因果链
  - 检测过度控制问题
  - 指出对系数估计的影响

### 第三步：生成报告

运行 `python main.py 论文.docx --report -e "Markdown报告" -o ./reviews`，生成：
- Markdown 格式评价报告
- 综合评分和等级
- 按优先级分类的修改建议

## 输出内容

### 综合评分
- 百分制得分
- 评级等级（优+/优/良+/良/中+/中/中-/不合格）

### 各维度详细评价
- 优点列表（引用原文）
- 问题列表（引用原文）
- 修改建议（按优先级）

### 修改建议（分类）
- **高优先级**：必须修改，影响论文合格
- **中优先级**：建议修改，提升论文质量
- **低优先级**：可选修改，完善细节

## 一票否决项

| 问题 | 后果 |
|-----|------|
| 参考文献<15篇 | 参考文献维度0分 |
| 外文文献<3篇（0篇扣5分，1篇扣3分，2篇扣1分） | 按缺少数目扣分 |
| 实证论文无稳健性检验 | 在"方法与论证严谨性"维度评价，不作硬性否决 |
| 实证论文存在明显过度控制 | 在"方法与论证严谨性"维度指出，不作硬性否决 |
| 重复率>25% | 综合评分不超过50分 |

## 评分等级标准

| 得分 | 等级 |
|-----|------|
| 90-100 | 优秀（优+） |
| 85-89 | 良好上（优） |
| 80-84 | 良好中（良+） |
| 75-79 | 良好下（良） |
| 70-74 | 中等上（中+） |
| 65-69 | 中等（中） |
| 60-64 | 中等下（中-） |
| <60 | 不合格 |

## 注意事项

1. 评价基于《经济与管理学院本科生论文指导要点》
2. 实证性论文必须检查稳健性检验
3. 修改建议具体可操作
4. 高优先级问题用**粗体**标注

## 依赖工具

- Python 3.8+
- Claude AI（深度内容评价）

## 论文类型判断

**实证性论文特征**：
- 包含回归分析、显著性检验
- 有数据、模型、变量
- 进行稳健性检验

**学理性论文特征**：
- 现状分析、问题分析、原因分析、对策建议
- 结构为"现状→问题→原因→对策"
